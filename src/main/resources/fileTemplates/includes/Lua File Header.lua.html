This is a Lua template similar to the a built-in Java template. It contains a code fragment that can be included into file templates (Templates tab) with the help of the #parse directive.
The template is editable. Along with the static text, code and comments, you can also use the predefined variables that will be then expanded like macros into the corresponding values.
Predefined variables will take the following values:
${PACKAGE_NAME}   name of the package in which the new class/interface is created
${USER}   current user system login name
${DATE}   current system date
${TIME}   current system time
${YEAR}   current year
${MONTH}   current month
${DAY}   current day of the month
${HOUR}   current hour
${MINUTE}   current minute
